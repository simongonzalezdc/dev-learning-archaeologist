## How the Specialist Uses This List

When building **The Plan** (Mode 3), for each topic in the curriculum:

1. **Match the topic to creator expertise:**
   - AI strategy, tooling evaluation, product thinking → **Nate B. Jones**
   - Type systems, validation, TypeScript → **Matt Pocock**
   - Agent architecture, folder systems, ICM → **Jake Van Cleef**
   - Agentic engineering, autonomous software, principled AI coding → **Indy Dev Dan**
   - Agent loops, context orchestration, Ralph Wiggum → **Geoffrey Huntley**

2. **Search for specific content:** Use the creator's YouTube channel search or known playlists to find a video that directly addresses the gap. Use their actual search URLs:
   - YouTube channel search: `https://www.youtube.com/@handle/search?query=[topic]`

3. **Only surface real content:** If a search returns no relevant video, do not invent one. State: "No verified content found from [creator] on this specific topic." Move to the next creator in the mapping.

4. **Citation format:**
   ```
   📺 [Creator Name] — "[Exact Video Title]"
   https://www.youtube.com/watch?v=[id]
   Why this fits: [one sentence connecting video to the specific gap]
   ```

5. **One recommendation per topic max.** The Plan does not become a reading list. It surfaces one targeted piece of verified content per curriculum item.
